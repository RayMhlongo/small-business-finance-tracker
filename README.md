# Small Business Finance Tracker

Static finance tracker for shops, freelancers, and service businesses. It runs from `index.html`, stores data in `localStorage`, exports/imports JSON backups, draws simple canvas charts, and creates PDFs with jsPDF.

## Deploy to GitHub Pages

1. Upload these files to the `small-business-finance-tracker` repository.
2. In GitHub, open **Settings > Pages**.
3. Set source to the `main` branch and root folder.

No backend or paid service is required.
